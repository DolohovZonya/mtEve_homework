from Bio import AlignIO

alignment = AlignIO.read("all_humans_mafft_alignment.fasta", "fasta")
root = alignment[-1].seq #the most ancient taxon stands in the end of the file
snp_max = 0
idshnik = ''
for record in alignment:
  snp_count = 0
  for i, base in enumerate(record.seq):
        if base != root[i]:
            snp_count += 1
        if snp_count > snp_max:
          snp_max = snp_count
          idshnik = record.id
        

print(f"max_distance: {snp_max, idshnik}")
According to the phylogenetic tree distances, we have 104 mutations between Lc1d1 (Central Africa) and B2 group (Argentinian). The mutation rate was taken from this paper: https://www.nature.com/articles/s41598-021-84583-1 as 3.12*10^-8 mutations per site per year (mean from 4,33 and 1,91). Thus, the mt Eve age can be estimated as 104 / (3.12*10^-8*16569), where 16569 is the mtGenome size. From this, mt Eve age is near 201,000 years (other halpogroups separated from africans around 201,000 years ago).
